HỌC CÁC KIẾN THỨC LẬP TRÌNH CƠ BẢN

1. Chú thích (Comment)

- Là phần mà trình biên dịch sẽ bỏ qua khi thực hiện chương trình.

- Mục đích của chú thích là để giải thích các đoạn code phức tạp, khó hiểu. Tuỳ vào ngôn ngữ lập trình mà có những cách sử dụng chú thích khác nhau.

VD:

// This is an inline comment


/* This is a block comment.

It can span multiple lines. */

2. Biến (variable)

- Là 1 nơi trong bộ nhớ có chức năng lưu trữ dữ liệu cần thiết trong quá trình chạy chương trình. 1 biến có thể là số, xâu, mảng … đây chính là các kiểu dữ liệu.

- Các kiểu dữ liệu cơ bản :

+ Số : Bao gồm số nguyên và số thực

VD : 5; 3,5; -4,5; ...

+ Xâu : Là một tập hợp các kí tự. Có thể bao gồm số, chữ, các kí tự đặc biệt …

VD : NguyenVanA, 56NguyenTrai, ...

+ Bool : Là kiểu dữ thể hiện sự đúng sai. Kiểu này chỉ có 2 giá trị là true và false.
+ Mảng : Là 1 tập hợp của nhiều giá trị, các giá trị trong mảng có thể thuộc cùng 1 loại kiểu dữ liệu hoặc không, tuỳ vào ngôn ngữ lập trình

VD : [“1”,”3”,”7”], [“NguyenVanA”,”PhamVanB”,”LeThiC”], ...

3. Toán tử

- Là công cụ dùng để thực hiện các phép tính hoặc chức năng với dữ liệu trong lập trình được thể hiện dưới dạng ký hiệu, biểu tượng nhằm thực hiện các biểu thức

- Toán tử số học : + (Cộng), - (Trừ), * (Nhân), / (Chia), % (Chia lấy dư) … Là các phép toán đơn giản trong xử lý số liệu

- Toán tử gán : dùng để gán / thay đổi giá trị cho biến. 1 số toán tử gán cơ bản

- Toán tử quan hệ : dùng để trả về mối quan hệ giữa hai biểu thức hoặc hai toán hạng. Kết quả trả về của nó là TRUE hoặc FALSE.

4. Câu lệnh điều kiện

- Là câu lệnh sẽ trả về kết quả khi thỏa mãn điều kiện của câu lệnh. Câu lệnh điều kiện gồm 2 phần. If và else. If (Nếu) là phần điều kiện đầu tiên, nếu dữ liệu thỏa mãn điều kiện này câu lệnh trong if sẽ được thực hiện và bỏ qua phần sau. Else (Nếu không thì) là phần điều kiện trái với điều kiện của phần if, sẽ được thực hiện nếu điều kiện trong if không thỏa mãn

- Cú pháp:

If (Điều kiện)

{

// Câu lệnh thực hiện nếu điều kiện đúng

}

else

{

// Thực hiện khi không thỏa mãn điều kiện

}

VD : Kiểm tra a là số chẵn hay lẻ

If (a % 2 == 0)

{

In ra a là số chẵn

}

else

{

In ra a là số lẻ

}

5. Câu lệnh lặp

- Dùng để chỉ thị cho máy thực thực hiện 1 công việc (1 câu lệnh hoặc 1 nhóm câu lệnh) nhiều lần mà không cần viết lại.

+ Lặp với số lần biết trước
+ Lặp với số lần không biết được
